<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/snowleopardAlexa/cats-video-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
